# ResFoodBackend

Backend layer of ResFood. Built with Express.js.

## Requirements
- Node.js

## Setup
1. Copy `.env.example` to `.env` or run directly (make sure `database.env` at root exists or use standard variables).
2. Install dependencies: `npm install`
3. Run the API: `npm run dev` (or `npm start`)

By default, the server runs on http://localhost:5000.
